# Clever-Bank
Kонсольное приложение для Clever-Bank.
Основные сущности:
- Банк
- Счёт
- Пользователь
- Транзакция

### Installation  
```
mvn clean install
```

### Run
1. Run docker app
2. Run [docker-compose.yaml](docker%2Fdocker-compose.yaml)
3. Run [CleverBank.java](src%2Fmain%2Fjava%2Fcom%2Fgithub%2Fkaydunov%2FCleverBank.java)
